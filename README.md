# Interactive Experience: Choose Your Own Path

This project is an interactive "Choose Your Own Adventure" style visual application developed in Processing about a regular anthropomorphic Argentinian bunny in his everyday life. You make choices and you live with them, and sometimes you fall into an unexpected bad ending. This was a good exercise for me to practice system flowcharts and how to design programmatic storylines.

## Technical Specifications
* **Environment:** Processing (Java)
* **Version Compatibility:** This sketch was originally developed using an older version of the software (**Processing 3**) in 2020. However, it has been tested and is fully compatible to run on the current release, **Processing 4**.

## Project Description
The application is an interactive experience where the user progresses through a daily routine by making different choices. The underlying code logic relies on screen state management, time-based transitions (utilizing the `millis()` function), and circular collision detection for the interactive user interface.

## How to Run
1. Download and install Processing 4.
2. Open the `sketch_elije_tu_propio_recorrido_processing.pde` file in the Processing IDE.
3. Press the **Run** button (play icon) to start the experience.
4. Follow the on-screen prompts and click to make your choices!
